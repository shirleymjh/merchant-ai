# 指标公式

- 订单优惠金额：SUM(discount_amt)，单位=元，来源字段=discount_amt，同义词=订单优惠金额、交易优惠金额、discount_amt
- 下单商品数量：SUM(sku_cnt)，单位=单，来源字段=sku_cnt，同义词=下单商品数量、订单商品数量、sku_cnt
- 商品总额分：SUM(product_amt)，单位=元，来源字段=product_amt，同义词=商品总额分、product_amt
- 运费金额分：SUM(freight_amt)，单位=元，来源字段=freight_amt，同义词=运费金额分、freight_amt
- 平台补贴金额：SUM(platform_subsidy_amt)，单位=元，来源字段=platform_subsidy_amt，同义词=平台补贴金额、platform_subsidy_amt
- 商家补贴金额：SUM(seller_subsidy_amt)，单位=元，来源字段=seller_subsidy_amt，同义词=商家补贴金额、seller_subsidy_amt
- 订单支付金额：SUM(pay_amt)，单位=元，来源字段=pay_amt，同义词=订单支付金额、交易支付金额、pay_amt
