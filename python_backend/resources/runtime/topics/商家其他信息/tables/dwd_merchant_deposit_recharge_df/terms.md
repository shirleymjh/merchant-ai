# 术语与同义词

- 创建时间：创建时间，同义词=创建时间、create_time，关联字段=create_time
- 修改时间：修改时间，同义词=修改时间、modify_time，关联字段=modify_time
- 商家id：商家id，同义词=商家id、merchant_id，关联字段=merchant_id
- 用户id：用户id，同义词=用户id、user_id，关联字段=user_id
- 补缴单单号(充值申请号)：补缴单单号(充值申请号)，同义词=补缴单单号(充值申请号)、deposit_recharge_id、保证金、押金，关联字段=deposit_recharge_id
- 交易流水号：交易流水号，同义词=交易流水号、trans_id，关联字段=trans_id
- 币种：币种，同义词=币种、currency，关联字段=currency
- 金额：金额, 通用单位元，同义词=金额、deposit_recharge_amt、金额, 通用单位元、保证金、押金，关联字段=deposit_recharge_amt
- 交易流水凭证：交易流水凭证，同义词=交易流水凭证、trans_voucher，关联字段=trans_voucher
- 备注：备注，同义词=备注、remark，关联字段=remark
- 日期分区yyyyMMdd：日期分区yyyyMMdd，同义词=日期分区yyyyMMdd、pt，关联字段=pt
- 保证金：商家保证金余额、冻结、充值或补缴流水，同义词=押金、保证金充值、保证金补缴
